## Tính năng
- 📊 Dashboard tổng quan với charts trực quan
- 🔴 Hệ thống cảnh báo tự động (Critical/High/Medium)
- 📋 Quản lý và gán nhãn reviews
- 🤖 Phân tích live (text + URL Shopee)
- 📥 Export CSV/Excel/PDF
- 🌙 Dark/Light mode

- **Frontend:** React + Vite
- **Backend:** FastAPI + SQLite
- **NLP:** PhoBERT (wonrax/phobert-base-vietnamese-sentiment)
- **Deploy:** Hugging Face Spaces (Docker)
